# CrunchyDL
Automated Script that uses yt-dlp to get videos given a queue from a txt file



How to setup: 
Install yt-dlp by downloading the exe file from the github repo
rename the exe file to yt.exe from yt-dlp.exe
Install ffmpeg and save it in the same place you saved yt.exe
Add the folder to environment variables under PATH
Edit uagent.txt and add links to queue.txt
finally run CrunchyDL-4.0.exe and follow the prompt
